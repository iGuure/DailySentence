DailySentence
=============
One day, one sentence.

Requirements
------------
+ Python 2.7
    - [Scrapy](http://scrapy.org "Scrapy official website") 1.0.3
+ Works on Linux, Windows, Mac OSX, BSD

Usage
-----
Run 'gendailysentence.py' to get today's meaningful sentence.
